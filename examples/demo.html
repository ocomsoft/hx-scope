hx-scope Extension Demo

Note: This demo uses a Service Worker to intercept requests and display the parameters being sent. You must serve this file from a web server (e.g., python -m http.server or npx serve) for the Service Worker to work. Opening the file directly (file://) will not work.

Demo 1: Basic Scoping

Two separate forms on the same page. Each button only submits inputs from its own scope.

Demo 2: Multiple Scopes

Inputs and buttons can belong to multiple scopes simultaneously.

Demo 3: Always-Included Inputs

Inputs without a scope are always included in all requests.
