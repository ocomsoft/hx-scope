hx-scope Extension Demos

hx-scope is an HTMX extension for selective input inclusion using CSS selectors.

The killer feature: Inputs with hx-name are excluded from parent form submission and only included when explicitly selected by hx-scope selectors.

Note: These demos use a Service Worker to intercept requests and display the parameters being sent. You must serve these files from a web server (e.g., python -m http.server) for the Service Worker to work.

1Basic CSS Class Selectors

Two separate forms on the same page. Each button uses a CSS selector to include only its inputs.

2Multiple Selectors (Comma-Separated)

Use comma-separated CSS selectors to include inputs from multiple groups.

3Attribute Selectors & Combining Selectors

Use attribute selectors and combine them with classes for powerful selection logic.

4Checkbox hx-off-value Feature

Use hx-off-value to send a specific value when checkboxes are unchecked.

5Component Isolation - Line Item Calculation

The Killer Feature! Multiple isolated components with reusable input names inside a single form.
